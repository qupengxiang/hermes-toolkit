## [0.1.0] - 2026-04-09

### Added

- Initial project setup with Git repository
- Complete SPEC.md with full功能规格说明书
- Multi-language support (i18n) with 4 languages:
  - Simplified Chinese (zh_CN)
  - English (en_US)
  - Japanese (ja_JP)
  - Traditional Chinese (zh_TW)
- Skills management module (UI framework)
- Memory management module (UI framework)
- Conversation history module (UI framework)
- Scheduled tasks (Cron) module (UI framework)
- Agent & API configuration module (UI framework)
- System status module (UI framework)
- Settings page with theme and language switching
- Comprehensive README.md and documentation

### Planned for 0.2.0

- Memory management full implementation
- Conversation history full implementation
- Data persistence layer
