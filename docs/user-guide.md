## 快速开始

### 启动程序

```bash
cd hermes-toolkit
python src/main.py
```

## Skills 管理

Skills 是 Hermes 的可扩展技能模块，类似于插件或脚本。

### 浏览 Skills

1. 点击左侧导航 **📁 Skills**
2. 左侧列表显示所有 Skills，按分类分组
3. 点击任意 Skill 可以查看详情

### 搜索和过滤

- **搜索框**：输入关键词实时过滤（匹配名称和内容）
- **分类下拉框**：按分类筛选 Skills

### 创建新 Skill

1. 点击右上角 **➕ 新建**
2. 填写表单：
   - **名称**：Skill 的唯一标识（英文，用横杠分隔）
   - **分类**：选择或输入新分类
   - **标签**：用逗号分隔多个标签
   - **描述**：简短描述 Skill 功能
   - **内容**：Markdown 格式的详细内容
3. 点击 **💾 保存**

### 编辑 Skill

1. 在列表中点击要编辑的 Skill
2. 修改表单中的内容
3. 点击 **💾 保存**

### 删除 Skill

1. 选中要删除的 Skill
2. 点击 **🗑️ 删除**
3. 在确认对话框中点击 **是**

### 导入 Skill

1. 点击右上角 **📥 导入**
2. 选择 `.md` 文件
3. Skill 会根据其 frontmatter 自动分类

### 导出 Skill

1. 选中要导出的 Skill
2. （后续版本支持）选择导出位置

## Memory 管理

Memory 是 Hermes 的持久记忆功能，用于存储跨会话的信息。

### 查看记忆

1. 点击左侧导航 **🧠 记忆**
2. 左侧列表显示所有记忆条目
3. 点击条目查看详细内容

### 记忆类型

- **👤 用户信息 (user)**：存储用户偏好设置等信息
- **🧠 系统记忆 (memory)**：存储系统级知识和上下文

### 创建记忆

1. 点击右上角 **➕ 新建**
2. 选择记忆类型
3. 输入记忆内容
4. 点击 **💾 保存**

### 编辑记忆

1. 在列表中点击要编辑的记忆
2. 修改内容
3. 点击 **💾 保存**

### 删除记忆

1. 选中要删除的记忆
2. 点击 **🗑️ 删除**
3. 在确认对话框中点击 **是**

### 搜索记忆

- 使用搜索框输入关键词，可以快速找到相关记忆

## 会话历史

## 定时任务

定时任务让你可以自动执行预设的 prompt，支持 Cron 表达式。

### 创建任务

1. 点击左侧导航 **⏰ 定时**
2. 点击右上角 **➕ 新建任务**
3. 填写表单：
   - **任务名称**：给任务起个名字
   - **执行周期**：Cron 表达式，如 `0 9 * * *`（每天早上9点）
     - 常用示例：
       - `every 1h` — 每小时
       - `every 30m` — 每30分钟
       - `0 9 * * *` — 每天9:00
       - `0 */2 * * *` — 每2小时
   - **通知方式**：
     - `当前对话` — 结果发送到当前聊天
     - `本地保存` — 保存到本地文件
     - `Telegram` — 通过 Telegram 发送
   - **执行内容**：要执行的 prompt
4. 点击 **💾 保存**

### 暂停/启用任务

1. 选中一个任务
2. 点击 **⏸ 暂停** 或 **▶ 启用** 按钮

### 编辑任务

1. 在列表中点击要编辑的任务
2. 修改表单内容
3. 点击 **💾 保存**

### 删除任务

1. 选中要删除的任务
2. 点击 **🗑️ 删除**
3. 在确认对话框中点击 **是**

## 智能体配置

### 添加智能体

1. 点击左侧导航 **🤖 智能体**
2. 点击右上角 **➕ 添加**
3. 填写表单：
   - **名称**：智能体显示名称，如 "Claude"
   - **提供商**：如 `anthropic`、`openai`、`openrouter`
   - **API 端点**：留空使用默认，或填入自定义地址
   - **API 密钥**：输入 API Key（会加密存储）
   - **默认模型**：如 `claude-sonnet-4-20250514`
   - **可用模型**：用逗号分隔多个模型名称
   - **系统提示词**：设定智能体的角色和行为
4. 点击 **💾 保存**

### 编辑智能体

1. 在列表中点击要编辑的智能体
2. 修改表单内容
3. 点击 **💾 保存**

### 删除智能体

1. 选中要删除的智能体
2. 点击 **🗑️ 删除**
3. 在确认对话框中点击 **是**

### 设为默认

1. 选中一个智能体
2. 点击 **⭐ 设为默认**
3. 默认智能体前会显示 ⭐ 标记

## 系统状态

### 查看状态

1. 点击左侧导航 **🔌 系统**
2. 查看以下信息：
   - **Hermes 运行状态**：是否连接
   - **版本号**：当前 Hermes 版本
   - **运行时间**：Hermes 持续运行时长
   - **MCP 服务**：已连接的 MCP 服务列表
   - **平台连接**：Telegram 等平台连接状态

### 快捷命令

- **重启 Hermes**：重启 Hermes 服务
- **重载配置**：重新加载配置文件
- **同步数据**：同步 Hermes 数据
- **健康检查**：诊断系统状态

## 设置

### 外观

- **主题**：切换深色/浅色模式

### 语言

- **语言切换**：简体中文 / English / 日本語 / 繁體中文
- 切换即时生效，无需重启

### 通知

- **任务完成通知**：开启/关闭任务完成提醒
- **任务失败通知**：开启/关闭任务失败提醒

### 路径配置

- **Skills 目录**：Skills 文件存放位置
- **Memory 文件**：Memory 数据文件位置
- **会话目录**：会话历史存放位置
- 点击 **浏览** 按钮选择新路径

## 常见问题

### Q: 启动时报错 "Module not found"

```bash
pip install -r requirements.txt
```

### Q: 界面显示乱码

确保系统安装了正确的中文字体。macOS 和 Linux 通常自带，Windows 可能需要额外安装。

### Q: Hermes 未运行提示

Hermes Toolkit 需要 Hermes AI 助手在后台运行。请先启动 Hermes。

### Q: 如何查看详细日志？

日志文件位于 `~/.hermes/logs/toolkit.log`
